The users stories we completed during this sprint include:

  1. As a user, I can access the website.
  2. As a user, I can visit the url at ‘/’ and see our homepage, which has our team logo, an unordered list of events, and a link to new events.
  3. As a user, I can visit your API at ‘/api/events’ and retrieve an object with key events containing all events in JSON format.
  4. As a user, I can search for events by submitting a GET request
  5. As a user, I can visit the url at /about and see information about our company
  6. As a user, I can visit a page for creating new events, which contains a form that is validated
